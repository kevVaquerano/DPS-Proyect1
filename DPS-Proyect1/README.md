# Gestor de Proyectos y Tareas

Sistema de gestión de proyectos y tareas desarrollado con React y Next.js.

## Enlace de despliegue

- **Frontend:** https://dps-proyect1.vercel.app
- **Backend (API):** https://dps-proyect1-production.up.railway.app

## Cómo ejecutar localmente

1. Clonar el repositorio desde GitHub
2. Entrar a la carpeta del proyecto: `cd DPS-Proyect1`
3. Instalar dependencias: `npm install`
4. En una terminal, iniciar el servidor de datos: `npm run server`
5. En otra terminal, iniciar el frontend: `npm run dev`
6. Abrir en el navegador: http://localhost:3000
